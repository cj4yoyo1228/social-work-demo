# 永續志工：阿明的成長數據飛輪 — 設計文件

> 日期：2026-02-22
> 狀態：已核准

## 專案概要

為永齡教育基金會製作一個「阿明的成長數據飛輪」互動展示頁面，展示志工服務如何透過 AI 轉譯成可量化的職場能力資產。整個頁面是一條引導式故事線，分三個階段呈現從「設定基準 → AI 行為轉譯 → 數據化資產」的完整數據生產線。

## 技術方案

| 項目     | 決定                                              |
| -------- | ------------------------------------------------- |
| 檔案     | `demo4-volunteer-flywheel.html`（單一 HTML 檔案） |
| 框架     | React 18 + ReactDOM（CDN）                        |
| JSX 編譯 | Babel Standalone（CDN）                           |
| 樣式     | Tailwind CSS（CDN）                               |
| 雷達圖   | SVG 手刻（不使用外部圖表庫）                      |
| 字體     | Noto Sans TC（Google Fonts）                      |
| 動畫     | CSS Keyframes + React state 驅動                  |

## 頁面流程（4 個主要狀態）

```
[頂部進度條：第一站 → 第二站 → 第三站]

狀態 1：第一站 — 職場指標對標 (3.1)
         ↓ 使用者點擊 [溝通力]
狀態 2：第二站 — 行為轉譯 & AI 追問 (3.2 & 3.3)
         ↓ 阿明回答 AI 追問
狀態 3：第三站 — 數據化資產 (3.4 & 3.5)
         ↓ 使用者點擊 [下載數位履歷]
狀態 4：證書視角 — 正式認證報告
```

## 各狀態詳細設計

### 狀態 1：第一站 — 職場指標對標 (3.1)

**目的：** 建立「設定基準」的專業感。

- 標題：「永齡職場力字典」
- 6 個能力卡片排成 2x3 網格：溝通力、領導力、問題解決、團隊合作、韌性、同理心
- 預設所有卡片呈現半透明/待選狀態
- 使用者點擊 **[溝通力]** → 卡片展開動畫，顯示：
  - 企業定義：有效傳遞資訊並達成共識
  - 永齡實作：能引導不同背景的學童進行情緒表達與衝突調解
- 展開後底部出現「前往第二站 →」按鈕

### 狀態 2：第二站 — 行為轉譯 & AI 追問 (3.2 & 3.3)

**目的：** 展示 AI 如何將日常日誌轉譯為職場能力指標。

- **左半部：** 阿明的日誌原文
  - 手寫風卡片：「今天小明跟小強吵架不寫功課，我很心累但還是先讓他們冷靜...」
  - 下方有「AI 轉譯中...」的脈衝動畫
  - 2 秒後動畫完成，顯示轉譯結果：「衝突解決 ✓」「情緒管理 ✓」高亮標記

- **右半部：** AI 對話框
  - 仿聊天介面的氣泡
  - AI 問：「阿明，面對這種挫折，你是如何調整策略的？」
  - 預設回答選項按鈕（免打字）：
    - 「我先讓自己冷靜，再想別的方法引導他們」
    - 「我請其他志工幫忙分開他們」
  - 使用者點擊任一選項 → 阿明的回答出現在對話中
  - 右側飛入動畫：[韌性指標] ↑15% + 成就勳章解鎖

- 底部：「前往第三站 →」按鈕

### 狀態 3：第三站 — 數據化資產 (3.4 & 3.5)

**目的：** 將前面的成長數據轉化為可視化的職場資產。

- **左側（60% 寬）：** SVG 動態雷達圖
  - 5 個頂點：溝通力、問題解決、韌性、同理心、領導力
  - 進場時從中心「綻放」動畫到最終數值
  - 使用者點擊 **[韌性]** 頂點 → 右側自動顯示行為證跡

- **右側（40% 寬）：** 數位履歷預覽
  - 行為證跡卡片：「第 8 週衝突處理回覆」（點擊韌性時出現）
  - 成就摘要：「服務期間展現卓越韌性，處理超過 10 次以上突發情緒危機」
  - 數據亮點：「問題解決能力 PR 92」

- **底部：** [下載永續背書之職場數位履歷] 大按鈕

### 狀態 4：證書視角

**目的：** 展示最終產出 — 具公信力的數位認證。

- 點擊下載 → 「生成中...」進度條動畫（3 秒）
- 動畫結束 → 畫面平滑切換為正式證書

- 證書佈局：
  - 頂部：永齡希望小學 + 「數位能力認證標章」
  - 中間：阿明頭像（圓形佔位圖）+ 姓名 + 18 週服務摘要
  - 核心：雷達圖轉為「職場競爭力報告」格式
  - 底部：[列印此證書] + [分享至 LinkedIn] 模擬按鈕

## 頂部導覽

- 固定在頁面頂部
- 左側：「← 回到首頁」連結（回 index.html）
- 中間：進度指示器（三個圓點 + 連線，當前階段高亮）
- 右側：「永續志工數據飛輪」標題

## 動畫規格

| 動畫         | 說明                  | 時間          |
| ------------ | --------------------- | ------------- |
| 卡片展開     | 溝通力卡片從小到大    | 0.4s ease-out |
| AI 轉譯中    | 脈衝/打字機效果       | 2s            |
| 韌性指標飛入 | 從右側滑入 + 數字跳動 | 0.6s          |
| 勳章解鎖     | 縮放 + 光暈           | 0.5s          |
| 雷達圖綻放   | 從中心展開到最終值    | 1.2s          |
| 進度條       | 線性進度從 0% 到 100% | 3s            |
| 狀態切換     | 淡出 + 淡入           | 0.3s          |

## 響應式設計

- 桌面版（>= 1024px）：雙欄佈局
- 平板/手機（< 1024px）：單欄堆疊

## 成功標準

1. 觀看者能感受到「從 3.1 到 3.5 是一條完整的數據生產線」
2. 每個階段的互動都直覺、不需要說明就能操作
3. 視覺上的專業感（品牌藍）和溫暖感（暖橘）共存
4. 證書視角能讓甲方感受到「可交付成果」的質感
